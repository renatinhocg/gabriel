# Projeto Gabriel

Este repositório contém três diretórios principais:
- `frontend`: aplicação React para usuários finais
- `admin`: painel administrativo em React
- `backend`: API e serviços do backend

## Como iniciar

1. Instale as dependências em cada pasta
2. Configure as variáveis de ambiente
3. Execute os serviços conforme documentação

---

Deploy será realizado via Railway e arquivos anexos (avatar/anexos) serão armazenados no AWS S3.
